# 🚀 HireSense AI

An AI-powered Resume Analyzer and ATS Optimization platform built with Python and Streamlit.

---

## ✨ Features

- 📄 Resume PDF Parsing
- 🎯 ATS Score Calculation
- 📊 Resume vs Job Description Similarity
- 💡 Skill Match Analysis
- 🤖 AI Resume Feedback (Gemini)
- ✨ AI Resume Rewriter
- 🎤 AI Interview Question Generator
- 📄 AI Cover Letter Generator
- 📥 Professional PDF Report
- 📜 Resume History (SQLite)
- 📈 Interactive Analytics Dashboard

---

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API
- Scikit-Learn
- Plotly
- SQLite
- ReportLab
- PyMuPDF
- pdfplumber

```bash

cd HireSense-AI

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
```
